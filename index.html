Описание Задания

Добро пожаловать на финальное задание курса! Ваша задача — выступить в роли инженера КИПиА и разработать базовый пакет проектной документации для тех. процесса, представленного на схеме P&ID ниже. Вы должны продемонстрировать умение читать схемы, извлекать необходимую информацию и оформлять ее в виде стандартных отраслевых документов.

Технологическая Схема (P&ID)

Внимательно изучите данную схему. Она является вашим основным источником информации для выполнения задания. Все приборы, которые необходимо задокументировать, находятся на этой P&ID.

Скачать PDF-версию P&ID

План Работы

  1. Анализ P&ID: Идентифицируйте все контрольно-измерительные приборы и клапаны, представленные на схеме.
  2. Разработка I/O List: Составьте полный перечень входов/выходов для всех приборов, которые подключаются к системе управления. Используйте предоставленный шаблон.
  3. Разработка Datasheets: Заполните технические паспорта (datasheets) для следующих всех приборов
    • Датчики уровня
    • Датчики давления
    • Регулирующий клапан
    • Датчики температуры
  4. Разработка Cable Schedule: Создайте кабельный журнал. Представьте, что все приборы со схемы подключаются к одной воображаемой клеммной коробке JB-9200.
  5. Прохождение Финального Квиза: После завершения работы над документацией, пройдите итоговый тест для проверки теоретических знаний по всему курсу.

Рекомендации по сдаче работы

Подготовьте пакет документов (I/O List, Datasheets, Cable Schedule), экспортировав их в формате CSV или PDF. Соберите все файлы в одну папку с вашим именем и отправьте на проверку. Удачи!
